# Minecraft Java + Bedrock (Geyser) Server

## Quick Start in Codespaces

1. Open this repo in Codespaces
2. Run:
   ```bash
   docker compose up -d
   ```
3. Wait for server to start (check logs with `docker compose logs -f`)

## Connection
- **Java**: localhost:25565
- **Bedrock**: localhost:19132

Replace `YOUR_USERNAME_HERE` in docker-compose.yml with your Minecraft username for OP.

Enjoy crossplay! 🌍⚒️
